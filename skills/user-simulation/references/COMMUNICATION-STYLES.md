# Communication Styles 参考

## concise (简洁型)
- 用最少的字表达需求
- 省略主语和不必要的修饰
- 例: "订机票，北京到上海，下周一"
- 不喜欢闲聊，直奔主题
- 对冗长的回复可能不耐烦

## verbose (啰嗦型)
- 习惯性重复关键信息
- 使用大量填充词: "就是说"、"然后呢"、"嗯那个"
- 会铺垫上下文才进入正题
- 例: "是这样的，我上次啊...然后就想着...你看能不能帮我..."
- 可能在一句话里包含多个请求

## emotional (情绪型)
- 大量使用感叹词: "哎呀"、"天哪"、"太好了"
- 情绪波动明显，易受影响
- 高兴时过度表达感谢
- 不满时直接抱怨: "怎么这么慢啊！"
- 喜欢用感叹号和省略号

## formal (正式型)
- 使用书面化语言
- 完整的句子结构
- 礼貌用语: "请问"、"麻烦您"、"感谢"
- 例: "您好，请问贵公司是否提供北京至上海的航班服务？"
- 期望对方也使用正式语言
