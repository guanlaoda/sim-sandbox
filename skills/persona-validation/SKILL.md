# 用户画像校验

## 概述

检查画像的丰化字段是否与骨架属性一致，以及各字段之间是否相互自洽。

## 校验清单

1. **年龄-职业一致性**：职业与年龄是否匹配合理？
2. **技术-语言一致性**：语言习惯是否与技术熟练度匹配？
3. **风格一致性**：性格描述是否与沟通风格匹配？
4. **触发-耐心一致性**：情绪触发点是否与耐心水平匹配？
5. **表达-画像一致性**：典型表达是否与整体画像匹配？

## 输出格式

返回 JSON 对象：
```json
{
  "is_valid": true,
  "score": 0.95,
  "issues": [
    {"field": "language_habits", "issue": "使用了技术术语但 tech_literacy 仅为 0.2", "severity": "high"}
  ],
  "suggestions": ["简化语言习惯以匹配较低的技术熟练度"]
}
```
